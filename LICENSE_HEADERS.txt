## Usage Instructions

### Automated License Header Tool
You can use tools like `licenseheaders` to automatically add these headers:

```bash
# Install licenseheaders
pip install licenseheaders

# Add headers to all source files
licenseheaders -t GPL3 -c "imnotUrban" -y 2024 -n "SAMI v2" -u "https://www.gnu.org/licenses/" -d . -x node_modules -x .git -x dist -x build
```

### Manual Addition
For manual addition, copy the appropriate header for your file type and paste it at the beginning of each source file.

### Exclusions
Do not add license headers to:
- Configuration files (package.json, tsconfig.json, etc.)
- Generated files
- Third-party dependencies
- Binary files
- Documentation files (README.md, docs/, etc.)
- Test fixtures and mock data
